# Assets Folder

Place your images, logos, and other media files here.

## Folder Structure

```
assets/
└── images/
    ├── logo.png (Add your square logo here)
    ├── founders/
    └── products/
```

## Logo Specifications

- **Format**: PNG, SVG, or JPG
- **Size**: At least 200x200px for square logo
- **Name**: `logo.png` (you can change this in Hero.jsx)
- **Background**: Transparent recommended for versatility

## Usage

Once you add your logo file (e.g., `logo.png`), it will be automatically referenced in:
- `src/components/Hero.jsx` - Hero section logo
- Any other components you want to use it in

Simply replace the filename in the import statement if you name it differently.
