# ARMB Landing Page

A modern, AI-first landing page for ARMB - transforming the future of EdTech and Business.

## 🎨 Design Features

- **Clean & Futuristic**: Minimal design with electric blue/neon cyan accents
- **Glassmorphism**: Modern blur and transparency effects
- **Smooth Animations**: Framer Motion animations throughout
- **Responsive**: Mobile-first, fully responsive design
- **AI-First Mindset**: Cutting-edge aesthetics and interactions

## 🚀 Tech Stack

- **React 18** - UI library
- **Tailwind CSS** - Utility-first styling
- **Framer Motion** - Advanced animations
- **Vite** - Lightning-fast build tool
- **Lucide React** - Beautiful icons

## 📁 Project Structure

```
src/
├── components/
│   ├── Hero.jsx          # Full-screen hero with animated blobs
│   ├── Products.jsx      # Product cards with hover effects
│   ├── Vision.jsx        # Company vision & mission
│   ├── Founders.jsx      # Team members showcase
│   ├── Contact.jsx       # Contact form
│   └── Footer.jsx        # Footer with links
├── pages/
│   └── Landing.jsx       # Main landing page
├── App.jsx               # App root
├── main.jsx              # Entry point
└── index.css             # Global styles & animations
```

## 🎯 Sections

1. **Hero** - Gradient headline with animated background blobs
2. **Products** - 4 AI solution cards with glow effects
3. **Vision** - Mission & vision statement
4. **Founders** - Team profiles (Akash Banerjee & Rakesh Mandal)
5. **Contact** - Contact form with validation
6. **Footer** - Social links and company info

## 🛠️ Installation & Setup

### Prerequisites
- Node.js 16+ 
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 📦 Key Dependencies

- `react@^18.2.0` - UI library
